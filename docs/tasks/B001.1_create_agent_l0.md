# Create Agent-L0: Principle Curator

**Status:** Completed

**Completed:** 2026-01-11  
**Parent:** B001 (Extensible Meta-Framework: Level Up Architecture)  
**Priority:** High

## Goal

Create the **forward-looking** Level 0 Principle Curator agent (`smaqit.L0`) that maintains framework purity for new principles and evolutionary changes. This agent assumes clean Level 0 files and prevents future contamination.

## Context

**What this is NOT:** A retroactive cleanup agent (that's `smaqit.L0.cleanup.agent.md`)

**What this IS:** A maintenance agent that:
- Reviews new principle additions for purity
- Validates principle refinements remain at L0
- Prevents directive contamination from entering framework files
- Guides users on what belongs at L0 vs L1

**Philosophy:** The agent assumes Level 0 is already pure and maintains that purity going forward through evolutionary development.

## Agent Responsibilities

### Input Scenarios

**Scenario 1: New Principle Addition**
- User: "Add principle: Idempotent Specifications"
- Agent: Reviews proposed principle, ensures pure form, adds to appropriate framework file

**Scenario 2: Principle Refinement**
- User: "Clarify Single Source of Truth principle"
- Agent: Refines existing principle narrative, maintains philosophical form

**Scenario 3: Contamination Prevention**
- User: "Add to SMAQIT.md: Agents MUST validate outputs"
- Agent: "This is a directive (L1), not a principle. Would you like me to help formulate the underlying principle, or shall I document this for Agent-L1?"

**Scenario 4: Principle Evolution**
- User: "Merge 'Layer Independence' and 'Prompt-Driven Requirements' principles"
- Agent: Consolidates principles while preserving conceptual clarity

### Core Behaviors

**Agent MUST:**
- Accept input about principles (additions, refinements, evolutions)
- Validate input is principle-form (WHY/WHAT conceptual)
- Reject directive-form input with guidance
- Maintain narrative, philosophical tone
- Preserve framework file structure and consistency
- Reference Task B001 for Level Up architecture context

**Agent MUST NOT:**
- Accept MUST/SHOULD/MUST NOT statements into L0
- Accept implementation details (paths, formats, commands)
- Accept procedural instructions (step-by-step workflows)
- Modify Level 1 templates or Level 2 agents
- Perform compilation to L1 (that's Agent-L1's job)

**Agent SHOULD:**
- Guide users when they provide directive-form input
- Suggest principle form when user intent is unclear
- Note when new principle implies L1 directives need updating
- Flag potential conflicts with existing principles

## Deliverable

**File:** `.github/agents/smaqit.L0.agent.md`

**Structure:** Follow development agent pattern (similar to `changelog.update.agent.md`)

**Sections:**
- Role — Forward-looking principle curator
- Framework Reference — Links to Task B001, SMAQIT.md
- Input — User requests about principles
- Output — Updated framework/*.md files
- Directives — MUST/MUST NOT/SHOULD for maintaining purity
- Principle Form Guidance — What constitutes pure principle
- Contamination Detection — How to identify directives disguised as principles
- Scope Boundaries — L0 only, no L1/L2 modification
- Completion Criteria — Principle added/refined in pure form
- Workflow Handover — Suggests Agent-L1 when directives need compilation

## Key Distinctions from Cleanup Agent

| Aspect | Cleanup Agent | Principle Curator (L0) |
|--------|--------------|------------------------|
| Assumption | L0 has contamination | L0 is pure |
| Direction | Retroactive (clean existing) | Forward-looking (maintain purity) |
| Input | Framework files to clean | User principle requests |
| Process | Extract directives, transform | Validate form, guide user |
| Output | Cleaned framework + extraction list | Updated framework in pure form |
| Handover | "Here's what to compile to L1" | "Consider L1 update if needed" |

## Notes

This agent is the **first** of three Level Up agents:
- **Agent-L0** (this task) — Maintains principle purity
- **Agent-L1** (future) — Compiles L0 principles to L1 directives
- **Agent-L2** (future) — Compiles L1 templates to L2 agents

The Cleanup Agent (`smaqit.L0.cleanup.agent.md`) is a one-time utility for addressing existing contamination. The Principle Curator is an ongoing maintenance tool.
